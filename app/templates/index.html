ABOUT US

Accidents occur quite often when mountainbiking - this is a given with such an extreme sport. Help can be difficult to get at the best of times, however it is extremely dificult when you are far into the mountains, sometimes alone and potentially unconsious. RideSafe was designed to get you help in hard to reach locations witout your input - automatically. RideSafe runs as a background service on your Android Phone and uses our machine learning algorithym to determine if you have had an accident. Once RideSafe determines you need help - A text message will be sent to your choosen emergency contact, containing your location.

RideSafe


Welcome to RideSafe. RideSafe was devoloped in order to tackle the unsuatibility of exsisting applications for riders of differing skills and disaplines. RideSafe adapts as you progress to minimise false positive detections. RideSafe's algorithym has been tried and tested, producing results greater than all other tested wearable based competators. Using machine learning, Ridesafe will only send an alert when you need it most and common drawbacks such as hard braking and dropping your phone wont trigger an alert. Originally devolopedas a proof of concept, Ridesafe is now the only mobile application using supervised machine learning in real-time for crash detection making it the most robust option avaible.

Behind the scenes

Android application written from scratch in Android studio using Java. Machine learning happens in real-time locally on each device running the app itself. The website you are now viewing serves mainily as an API (Login protected access) for RideSafe. The site is currently written in python3/html. The Flask application runs on a custom built arm64 docker container running from my home on a RaspberryPi.
